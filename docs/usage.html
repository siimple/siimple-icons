Installation
Import siimple-icons using a CDN
Simply add the following link tag in the head of each page that you want to use siimple-icons on: 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/siimple-icons@{{ site.version }}/dist/siimple-icons.min.css">
You can browse the source of the package at https://cdn.jsdelivr.net/npm/siimple-icons/.
Install siimple-icons as a package
You can easily install the latest version of siimple-icons via npm or yarn. 
$ npm install siimple-icons
$ yarn add siimple-icons
Now you should add a reference to siimple-icons.min.css file in the head tag of each HTML page where you are going to use siimple-icons: 
<link rel="stylesheet" href="./node_modules/siimple-icons/dist/siimple-icons.min.css">
Usage
Using CSS styles
You can use siimple-icons in your project adding a <span> tag (or a <i> tag for brevity) and reference the icon using the following class names: si to add the icon base style and si- followed by the icon name. 
<!-- Add a left arrow icon -->
<span class="si si-arrow-left"></span>

<!-- Add a star icon -->
<span class="si si-star"></span>
All icons will automatically inherit the font-size and the color style rules. 
<span class="si si-plus" style="color: red; font-size:20px;"></span>
Using SVG sprites
You can use our icons strpite in svg format creating a <svg> tag and calling the desired icon using a <use> tag. 
<svg>
    <use xlink:href="./node_modules/siimple-icons/dist/siimple-icons.svg#arrow-left"></use>
</svg>
You can also style your icon adding a style attribute to your <svg> tag. 
<svg style="color: blue; font-size: 20px">
    <use xlink:href="./node_modules/siimple-icons/dist/siimple-icons.svg#calendar"></use>
</svg>
